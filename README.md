# 開発

- nodejs: 7.5.0
- npm: 4.1.2

```sh
git clone git@github.com:otiai10/kanColleWidget.git
cd kanColleWidget
npm install && npm start
```

- `npm run build` きれいなjsをビルドするやつ
- `npm start` 編集差分を見てビルドするやつ
- `npm release` **release/kcwidget** にリリースビルドをつくるやつ
